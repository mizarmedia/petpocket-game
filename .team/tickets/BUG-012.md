# BUG-012: Missing ARIA labels on action buttons

**Severity:** High
**Status:** Fixed
**Found by:** audit
**Assigned to:** generalist-2
**Created:** 2025-12-25 15:41

## Description
Action buttons (Feed, Play, Sleep, Clean, Hatch New Pet) are missing aria-label attributes.

Steps to reproduce:
1. Navigate to main game screen
2. Inspect action buttons in DevTools
3. Check for aria-label attributes

Expected: All buttons should have descriptive aria-label attributes for screen readers
Actual: Action buttons have no aria-label (Header buttons are correctly labeled)

Affected buttons:
- Feed button
- Play button  
- Sleep button
- Clean button
- Hatch New Pet button

Impact: Screen reader users cannot understand button purposes
Severity: High - Accessibility issue affecting usability

Related: AUDIT_REPORT.md MEDIUM-004 (partially fixed)
